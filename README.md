\# ExpenseFlow AI 🚀



ExpenseFlow AI is an expense tracking and bookkeeping platform designed to help freelancers and small businesses manage their finances easily.



\## Features



✅ Add, edit, and delete expenses  

✅ Dashboard with expense summaries  

✅ Category-based expense tracking  

✅ Monthly reports  

✅ Search and sorting  

✅ CSV export  

✅ Accounting dashboard  



\## Technology Stack



\### Frontend

\- React

\- Vite

\- JavaScript

\- CSS



\### Backend

\- Node.js

\- Express.js

\- SQLite Database
